# eslint-config-tjw-base

The Jared Wilcurt's base ESLint rules


## Using this

1. `npm install --save-dev eslint eslint-config-tjw-base`
1. In your `.eslintrc.js` add `tjw-base` to your `extends` like so:
    ```js
    module.exports = {
      'extends': [
        'tjw-base'
      ]
    };
    ```
